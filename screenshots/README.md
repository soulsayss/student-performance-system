# Screenshots

Add your project screenshots here!

## How to Add Screenshots

1. Take screenshots of your application:
   - Landing page
   - Student dashboard
   - Teacher dashboard
   - ML predictions page
   - Career guidance page
   - Analytics/reports

2. Save them with these names:
   - `landing.png`
   - `student-dashboard.png`
   - `predictions.png`
   - `career.png`
   - `analytics.png`

3. Copy the images to this folder

4. They will automatically appear in your README.md!

## Tips for Good Screenshots

- Use full screen or maximize window
- Show actual data (not empty screens)
- Capture both light and dark mode if possible
- Use high resolution (1920x1080 or higher)
- Crop out unnecessary parts
- Highlight important features

## Optional: Use a Screenshot Tool

- Windows: Snipping Tool or Win + Shift + S
- Mac: Cmd + Shift + 4
- Linux: Screenshot app or Flameshot

After adding screenshots, commit and push:
```bash
git add screenshots/
git commit -m "Add project screenshots"
git push
```
